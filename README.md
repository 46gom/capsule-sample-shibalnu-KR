## Overview

이 캡슐은 외부 API로부터 이미지 URL을 받아와 이미지를 보여줍니다. 
해당 캡슐을 통하여 외부 API로부터 데이터를 받는 방법, view 및 layout을 사용하는 방법을 알 수 있습니다.
